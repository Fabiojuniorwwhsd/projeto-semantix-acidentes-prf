# Projeto Semantix / EBAC — Acidentes em Rodovias Federais

## Descrição
Este projeto tem como objetivo analisar acidentes em rodovias federais brasileiras entre 2022 e 2024, utilizando dados públicos da Polícia Rodoviária Federal (PRF). A proposta é identificar padrões, fatores de risco e principais insights que possam apoiar ações de prevenção e tomada de decisão.

## Problema analisado
O projeto busca entender a evolução dos acidentes, mortes e feridos graves em rodovias federais, além de identificar:
- estados com maior concentração de acidentes
- causas mais frequentes
- períodos do dia com maior ocorrência
- padrões gerais ligados à gravidade dos acidentes

## Fontes de dados
- Polícia Rodoviária Federal (PRF) — dados abertos de acidentes agrupados por ocorrência
- Arquivos utilizados: 2022, 2023 e 2024

## Ferramentas utilizadas
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Google Looker Studio
- GitHub

## Estrutura do repositório
- `docs/` → documentação do projeto
- `notebooks/` → notebook da análise exploratória
- `scripts/` → script Python da análise
- `exports/` → arquivos resumo gerados na EDA

## Principais achados
- Houve crescimento no número de acidentes entre 2022 e 2024
- Também houve aumento em mortes e feridos graves
- Minas Gerais, Santa Catarina e Paraná aparecem entre os estados com mais acidentes
- Causas ligadas ao comportamento do condutor aparecem com forte relevância
- A faixa entre 12h e 17h concentra mais acidentes
